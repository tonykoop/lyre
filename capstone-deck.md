# Capstone Deck

## Slide 1 - Title

Lyre
Historically informed Greek yoke lyre with 13-string performance variant and
family-aware SolidWorks workflow.

## Slide 2 - What This Project Is

An original chelys/yoke-lyre design packet rooted in ancient Greek instrument
grammar and modernized through ergonomics, joinery, and parametric CAD.

## Slide 3 - File Map

- `design.md`
- `reverse-engineering.md`
- `family-spec.csv`
- `cad/`
- `cnc/`
- `wolfram/`
- `validation.csv`

## Slide 4 - Build Workflow

Reference evidence -> concept brief -> family spec -> SW globals/equations ->
master layout -> mule -> validation -> refined build.

## Slide 5 - Family Strategy

Four starting configurations:

- `LYR-07-HERMES`
- `LYR-13-MULE`
- `LYR-13-WANDERER`
- `LYR-13-WANDERER-ERG`
