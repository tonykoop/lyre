# Drawing Brief

The eventual drawing packet should communicate:

- front, side, top, back, and section views of the lyre family;
- all master envelope dimensions with explicit note that first-pass targets are
  not measured archaeological dimensions;
- arm socket joinery, yoke saddle, and lashing groove details;
- bridge, tailpiece, peg, and string-path details;
- optional ergonomic features as suppressible/configurable elements;
- optional pickup and strap interfaces as reversible accessory systems;
- safe inlay zones and prohibited high-motion acoustic zones;
- stand/cradle joinery details.

Separate sheets should exist for:

- historical-clean configuration;
- ergonomic performance configuration;
- mule/prototype-only simplifications.
