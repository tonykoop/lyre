Instrument Maker Build Log

Lyre

An original historically informed Greek yoke lyre with a 13-string performance build, family-aware SolidWorks workflow, and Japanese joinery-inspired detailing.

What This Is

This project reinterprets the ancient Greek chelys/yoke lyre through a modern but disciplined design language. The packet preserves ancient topology and playing logic while exploring ergonomic seated support, optional pickup readiness, and configurable joinery for arms, yoke, and shell construction.

The Family

  • LYR-07-HERMES: historical reference
  • LYR-13-MULE: structural and acoustic prototype
  • LYR-13-WANDERER: main build target
  • LYR-13-WANDERER-ERG: ergonomic performance variant

Build Workflow

Reference evidence feeds the concept brief, then the family spec, then the SolidWorks equations and design table, then the mule build, then validation, then the refined visual build.
